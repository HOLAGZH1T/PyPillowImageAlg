# Pillow Image Collage Tools

A small Python image-processing project using Pillow.

The project contains scripts for creating image collages and mosaic-like compositions from local image files. It started as an experiment with Python-based image manipulation and is used as a learning project for file handling, image processing and simple automation.

## Features

- Load and process local image files
- Resize and arrange images
- Create simple image collages
- Generate mosaic-style image compositions
- Use Pillow for image manipulation
